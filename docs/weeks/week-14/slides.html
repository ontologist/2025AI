Week 8 Review: Clustering & AI Overview

第8週復習: クラスタリング・AI概論

Key Concepts:

  • Unsupervised Learning: Finding patterns without labels
  • K-means Algorithm:
    1. Choose K (number of clusters)
    2. Initialize centroids
    3. Assign points to nearest centroid
    4. Update centroids
    5. Repeat until convergence
  • AI Hierarchy: AI ⊃ ML ⊃ DL ⊃ Neural Networks
  • Applications: Netflix recommendations, Spotify playlists

主要概念:

  • 教師なし学習: ラベルなしでパターンを発見
  • K-meansアルゴリズム:
    1. K（クラスタ数）を選択
    2. 重心を初期化
    3. ポイントを最も近い重心に割り当て
    4. 重心を更新
    5. 収束まで繰り返し
  • AI階層: AI ⊃ ML ⊃ DL ⊃ ニューラルネットワーク
  • 応用: Netflix推薦、Spotifyプレイリスト

Week 9 Review: AI Types & ML Paradigms

第9週復習: AIのタイプ・MLパラダイム

Types of AI

AIのタイプ

  • Narrow AI (ANI): Specific tasks (current)
  • General AI (AGI): Human-level (future)
  • Super AI (ASI): Beyond human (speculative)

3 ML Paradigms

3つのMLパラダイム

  • Supervised: Labeled data → predictions
  • Unsupervised: Find patterns
  • Reinforcement: Trial & error with rewards

Real Examples: Gmail (supervised), Amazon (unsupervised), AlphaGo (RL)

実例: Gmail（教師あり）、Amazon（教師なし）、AlphaGo（強化学習）

Week 10 Review: Supervised Learning Basics

第10週復習: 教師あり学習基礎

Core Concepts:

  • Classification: Predict discrete categories
  • Regression: Predict continuous values
  • Training Data Split:
    • Training: 70-80%
    • Validation: 10-15%
    • Test: 10-15%
  • Overfitting: Model memorizes training data
  • Underfitting: Model too simple

Applications: Credit fraud detection, house price prediction

中核概念:

  • 分類: 離散カテゴリを予測
  • 回帰: 連続値を予測
  • 訓練データ分割:
    • 訓練: 70-80%
    • 検証: 10-15%
    • テスト: 10-15%
  • 過学習: モデルが訓練データを記憶
  • 学習不足: モデルが単純すぎる

応用: クレジット不正検出、住宅価格予測

Week 11 Review: Classification Algorithms

第11週復習: 分類アルゴリズム

Decision Trees

決定木

  • Tree structure with decision nodes
  • ツリー構造と決定ノード
  • Easy to interpret
  • 解釈しやすい
  • Prone to overfitting
  • 過学習しやすい

Logistic Regression

ロジスティック回帰

  • Outputs probabilities (0-1)
  • 確率を出力（0-1）
  • Linear decision boundary
  • 線形決定境界
  • Fast and simple
  • 高速でシンプル

Support Vector Machines (SVM) / サポートベクターマシン

  • Finds maximum margin hyperplane / 最大マージン超平面を見つける
  • Effective in high dimensions / 高次元で効果的

Week 12 Review: Neural Networks

第12週復習: ニューラルネットワーク

Key Components:

  • Layers: Input → Hidden → Output
  • Neurons: Processing units
  • Weights: Connection strengths
  • Activation Functions: ReLU, Sigmoid, Tanh

Gradient Descent:

  • Optimization algorithm
  • Iteratively adjusts weights to minimize error
  • Learning rate controls step size
  • Types: Batch, SGD, Mini-batch

主要コンポーネント:

  • 層: 入力 → 隠れ → 出力
  • ニューロン: 処理ユニット
  • 重み: 接続の強度
  • 活性化関数: ReLU、シグモイド、Tanh

勾配降下法:

  • 最適化アルゴリズム
  • エラーを最小化するために重みを反復的に調整
  • 学習率がステップサイズを制御
  • タイプ: バッチ、SGD、ミニバッチ

Week 13 Review: Reinforcement Learning

第13週復習: 強化学習

RL Framework:

  • Agent: Learner/decision maker
  • Environment: What agent interacts with
  • Actions: What agent can do
  • Rewards: Feedback signals
  • Policy: Strategy for choosing actions

N-Armed Bandit:

  • Foundation RL problem
  • Balance exploration vs exploitation
  • ε-Greedy strategy: Explore with probability ε

RLフレームワーク:

  • エージェント: 学習者/意思決定者
  • 環境: エージェントが相互作用するもの
  • 行動: エージェントができること
  • 報酬: フィードバック信号
  • ポリシー: 行動を選択する戦略

N本腕バンディット:

  • 基礎的なRL問題
  • 探索と活用のバランス
  • ε-貪欲戦略: 確率εで探索

Quick Reference: ML Paradigms

クイックリファレンス: MLパラダイム

Supervised Learning / 教師あり学習: ✓ Has labeled training data ✓ Goal: Predict labels for new data ✓ Examples: Classification, Regression ✓ Apps: Spam filter, house prices Unsupervised Learning / 教師なし学習: ✓ No labels needed ✓ Goal: Find hidden patterns ✓ Examples: Clustering, Dimensionality reduction ✓ Apps: Customer segmentation, recommendations Reinforcement Learning / 強化学習: ✓ Learns through interaction ✓ Goal: Maximize cumulative reward ✓ Examples: Bandit problems, game AI ✓ Apps: A/B testing, robotics, games

Important Formulas / 重要な公式

You Should Know:
