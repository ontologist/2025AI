Week 6: Introduction to Machine Learning

第6週: 機械学習入門

Core ML Concepts / MLの中核概念:

Machine Learning = Learning from Data 機械学習 = データからの学習 Traditional Programming: Rules + Data → Answer 従来のプログラミング: ルール + データ → 答え Machine Learning: Data + Answer → Rules (learned) 機械学習: データ + 答え → ルール（学習）

Three Types of ML:

  • Supervised: Learn from labeled examples
  • Unsupervised: Find patterns without labels
  • Reinforcement: Learn through trial and error

MLの3つのタイプ:

  • 教師あり: ラベル付き例から学習
  • 教師なし: ラベルなしでパターンを発見
  • 強化学習: 試行錯誤を通じて学習

Week 7: Supervised Learning Basics

第7週: 教師あり学習基礎

The ML Workflow / MLワークフロー:

1. Collect Data → 2. Prepare Data → 3. Train Model → 4. Evaluate → 5. Predict データ収集 データ準備 モデル訓練 評価 予測

Classification / 分類

  • Predicting categories
  • カテゴリの予測
  • Example: Spam vs Not Spam
  • 例: スパム vs 非スパム
  • Output: Discrete labels
  • 出力: 離散ラベル

Regression / 回帰

  • Predicting continuous values
  • 連続値の予測
  • Example: House prices
  • 例: 住宅価格
  • Output: Numeric values
  • 出力: 数値
