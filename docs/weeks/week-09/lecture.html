Key Definition / ä¸»è¦å®šç¾©

Artificial Intelligence (AI): The science of creating machines and software that can perform tasks that typically require human intelligence, such as visual perception, speech recognition, decision-making, and language translation.

The Foundations of AI

Artificial Intelligence is built on several foundational concepts:

1. Reasoning and Problem Solving

AI systems can analyze situations, evaluate options, and make logical decisions. This includes:

  • Search algorithms (finding optimal solutions)
  • Game playing (chess, Go, poker)
  • Planning and scheduling
  • Constraint satisfaction

2. Knowledge Representation

Storing and organizing information in ways machines can use:

  • Semantic networks
  • Ontologies and knowledge graphs
  • Rules and logic systems
  • Databases and data structures

3. Learning

The ability to improve performance through experience:

  • Pattern recognition
  • Adaptation to new situations
  • Generalization from examples
  • Self-improvement over time

4. Natural Language Processing

Understanding and generating human language:

  • Text analysis and generation
  • Machine translation
  • Sentiment analysis
  • Chatbots and virtual assistants

5. Perception

Interpreting sensory input:

  • Computer vision (image and video analysis)
  • Speech recognition
  • Object detection and tracking
  • Facial recognition

3. Reinforcement Learning (å¼·åŒ–å­¦ç¿’)

Definition: Learning to make decisions by taking actions in an environment to maximize cumulative reward.

Key Characteristics:

  • Agent: The learner/decision maker
  • Environment: What the agent interacts with
  • Actions: Choices the agent can make
  • Rewards: Feedback from environment
  • Goal: Maximize total reward over time

How It Works:

  1. Agent observes current state
  2. Agent takes an action
  3. Environment provides reward
  4. Environment transitions to new state
  5. Agent learns from experience
  6. Repeat

Applications:

  • Game Playing: AlphaGo, Atari games
  • Robotics: Robot navigation and control
  • Autonomous Vehicles: Self-driving cars
  • Resource Management: Energy optimization
  • Finance: Trading strategies

Common Algorithms:

  • Q-Learning
  • Deep Q-Networks (DQN)
  • Policy Gradient Methods
  • Actor-Critic
  • Proximal Policy Optimization (PPO)

TEST QUESTION 6: Short Answer

Explain the difference between classification and regression in supervised learning. Give one example of each.

Sample Answer:

Classification predicts categorical labels (discrete classes). The output is a category or class label. Example: Predicting whether an email is "spam" or "not spam" - there are distinct categories.

Regression predicts continuous numerical values. The output can be any value within a range. Example: Predicting house prices - the output can be any dollar amount like $250,000 or $375,500.

The key difference is that classification outputs discrete categories while regression outputs continuous numbers.

TEST QUESTION 7: Short Answer

Describe how reinforcement learning works. Include the main components and the learning process.

Sample Answer:

Reinforcement learning involves an agent learning to make decisions through trial and error in an environment.

Main components:

  • Agent: The learner/decision maker
  • Environment: The world the agent interacts with
  • State: Current situation
  • Action: What the agent can do
  • Reward: Feedback from the environment

Learning process: The agent observes the current state, takes an action, receives a reward (positive or negative), and transitions to a new state. Through many iterations, the agent learns which actions maximize cumulative rewards over time. Example: AlphaGo learned to play Go by playing millions of games and receiving rewards for winning.

TEST QUESTION 10: Essay Question

Compare and contrast the three main types of machine learning (supervised, unsupervised, and reinforcement learning). For each type, explain:

  • What kind of data it uses
  • What it learns to do
  • One real-world application
  • One advantage and one limitation

Sample Answer:

Supervised Learning:

Data: Uses labeled data with known correct outputs (X â†’ Y pairs)

Learning Goal: Learn to predict labels for new data by finding patterns in labeled examples

Application: Email spam filtering - emails are labeled as spam/not spam, and the system learns to classify new emails

Advantage: High accuracy when sufficient labeled data is available

Limitation: Requires expensive and time-consuming data labeling

Unsupervised Learning:

Data: Uses unlabeled data with no predefined categories

Learning Goal: Discover hidden patterns, structures, and groupings in data

Application: Customer segmentation - grouping customers by behavior without predetermined categories

Advantage: No labeling required, can discover unexpected patterns

Limitation: Evaluation is subjective, results may be difficult to interpret

Reinforcement Learning:

Data: Generates its own data through interaction with environment

Learning Goal: Learn optimal actions to maximize cumulative rewards

Application: Game playing (AlphaGo) - learning winning strategies through trial and error

Advantage: Can learn complex behaviors without explicit examples

Limitation: Requires many iterations and can be computationally expensive

Comparison: All three are machine learning approaches, but they differ in how they learn. Supervised learning needs a teacher (labeled data), unsupervised learning explores independently, and reinforcement learning learns through trial and error with rewards.
