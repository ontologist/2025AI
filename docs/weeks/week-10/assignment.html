Week 10 Assignment

第10週課題

Assignment Title:

Supervised Learning Problem Analysis

課題タイトル:

教師あり学習問題分析

Due Date: Before next class (Week 11)

Estimated Time: 45-55 minutes

Grade Weight: Part of 40% Individual Reports

締切: 次回授業前（第11週）

予想時間: 45-55分

成績配分: 個別レポート40%の一部

Learning Objectives / 学習目標

  • Distinguish between classification and regression problems
  • Identify appropriate features and labels for supervised learning
  • Understand the importance of data splitting
  • Recognize overfitting and underfitting issues
  • 分類問題と回帰問題を区別する
  • 教師あり学習に適切な特徴とラベルを特定する
  • データ分割の重要性を理解する
  • 過学習と学習不足の問題を認識する

Part 1: Classification vs Regression (30 points)

パート1: 分類 vs 回帰（30点）

Question 1.1 (20 points)

For each of the following scenarios, determine whether it is a classification or regression problem, and explain your reasoning:

  1. Predicting whether a loan applicant will default on their loan
  2. Estimating the number of customers who will visit a store next week
  3. Determining if an email is spam, promotional, or personal
  4. Forecasting tomorrow's maximum temperature
  5. Identifying which genre a book belongs to
  6. Predicting a student's final exam score based on homework grades
  7. Detecting whether a medical image shows cancer or not
  8. Estimating how long a project will take to complete

For each, also specify:

  • What would be the output (discrete class or continuous value)?
  • Provide ONE example of what a prediction might look like

Format as a table or structured list.

質問1.1（20点）

以下の各シナリオについて、それが分類問題か回帰問題かを判断し、推論を説明してください:

  1. ローン申請者がローンでデフォルトするかどうかを予測する
  2. 来週店舗を訪問する顧客数を推定する
  3. メールがスパム、プロモーション、または個人的なものかを判断する
  4. 明日の最高気温を予測する
  5. 本が属するジャンルを特定する
  6. 宿題の成績に基づいて学生の最終試験スコアを予測する
  7. 医療画像が癌を示しているかどうかを検出する
  8. プロジェクトが完了するまでにかかる時間を推定する

それぞれについて、以下も指定してください:

  • 出力は何か（離散クラスまたは連続値）？
  • 予測がどのように見えるかの例を1つ提供する

表または構造化されたリストとして形式化してください。

Question 1.2 (10 points)

Consider a scenario where you want to predict student performance. Could this be formulated as BOTH a classification and regression problem? Explain:

  • How would you frame it as a classification problem?
  • How would you frame it as a regression problem?
  • What are the advantages and disadvantages of each approach?
  • Which approach would you choose and why?

Write 150-200 words.

質問1.2（10点）

学生のパフォーマンスを予測したいシナリオを考えてください。これは分類問題と回帰問題の両方として定式化できますか？説明してください:

  • どのように分類問題として組み立てるか？
  • どのように回帰問題として組み立てるか？
  • 各アプローチの利点と欠点は何か？
  • どのアプローチを選択し、なぜですか？

150-200語で記述してください。

Part 2: Feature and Label Identification (30 points)

パート2: 特徴とラベルの特定（30点）

Scenario: Restaurant Review Rating Prediction

You work for a food delivery app and want to build a system that predicts restaurant ratings (1-5 stars) based on various factors to help users discover good restaurants.

シナリオ: レストランレビュー評価予測

あなたは食品配達アプリで働いており、ユーザーが良いレストランを発見するのを助けるために、さまざまな要因に基づいてレストラン評価（1-5つ星）を予測するシステムを構築したいと考えています。

Question 2.1 (15 points)

For this scenario:

  1. Is this a classification or regression problem? Justify your answer. (Could it be both?)
  2. Identify at least SIX features that would be useful for prediction. For each feature:
    • State whether it's numerical or categorical
    • Explain why it would be relevant
    • Give an example value
  3. What is the label/target variable?
  4. What challenges might you face with this dataset?

Present in clear, organized format (200-250 words).

質問2.1（15点）

このシナリオについて:

  1. これは分類問題ですか、それとも回帰問題ですか？答えを正当化してください。（両方になり得ますか？）
  2. 予測に有用な少なくとも6つの特徴を特定してください。各特徴について:
    • 数値かカテゴリかを述べる
    • なぜそれが関連性があるかを説明する
    • 例の値を示す
  3. ラベル/ターゲット変数は何ですか？
  4. このデータセットでどのような課題に直面する可能性がありますか？

明確で整理された形式で提示してください（200-250語）。

Question 2.2 (15 points)

Data Splitting Analysis:

You have 10,000 restaurant reviews with ratings. Explain how you would split this data:

  • What proportions would you use for training, validation, and test sets? Why?
  • Why is it important NOT to use test data during training?
  • What is the purpose of the validation set?
  • How would you ensure the split is representative (e.g., balanced across ratings)?
  • What problems could arise from a poor data split?

Write 200-250 words with clear explanations.

質問2.2（15点）

データ分割分析:

評価付きの10,000のレストランレビューがあります。このデータをどのように分割するかを説明してください:

  • 訓練、検証、テストセットにどのような割合を使用しますか？なぜですか？
  • 訓練中にテストデータを使用しないことがなぜ重要ですか？
  • 検証セットの目的は何ですか？
  • 分割が代表的であることをどのように確保しますか（例: 評価全体でバランスが取れている）？
  • 不適切なデータ分割からどのような問題が生じる可能性がありますか？

明確な説明で200-250語で記述してください。

Part 3: Overfitting and Underfitting (20 points)

パート3: 過学習と学習不足（20点）

Question 3.1 (20 points)

Consider these two scenarios:

Scenario A: A spam email classifier achieves 99% accuracy on training data but only 60% on test data.

Scenario B: A house price predictor achieves 65% accuracy on both training and test data.

Answer the following:

  1. Which scenario shows overfitting? Which shows underfitting? Explain your reasoning. (100 words)
  2. For Scenario A:
    • What likely caused this problem?
    • Propose THREE specific solutions to fix it
    • How would you know if your solutions worked?
    (100 words)
  3. For Scenario B:
    • What might be causing the poor performance?
    • Propose THREE specific solutions to improve it
    • Would collecting more data help? Why or why not?
    (100 words)

Total: 300 words with specific, actionable recommendations.

質問3.1（20点）

次の2つのシナリオを考えてください:

シナリオA: スパムメール分類器は訓練データで99%の精度を達成しますが、テストデータではわずか60%です。

シナリオB: 住宅価格予測器は訓練データとテストデータの両方で65%の精度を達成します。

以下に答えてください:

  1. どのシナリオが過学習を示していますか？どれが学習不足を示していますか？推論を説明してください。（100語）
  2. シナリオAについて:
    • この問題を引き起こした可能性が高いのは何ですか？
    • それを修正するための3つの具体的な解決策を提案してください
    • 解決策が機能したかどうかをどのように知りますか？
    （100語）
  3. シナリオBについて:
    • パフォーマンスの低さを引き起こしている可能性があるのは何ですか？
    • それを改善するための3つの具体的な解決策を提案してください
    • より多くのデータを収集することは役立ちますか？なぜ、またはなぜそうではないのですか？
    （100語）

合計: 具体的で実行可能な推奨事項で300語。

Bonus Question (Optional, +10 points)

ボーナス質問（任意、+10点）

Real-World Application Design

Design a supervised learning system for a problem you care about (e.g., education, health, environment, social issues). Describe:

  • The problem you're solving
  • Whether it's classification or regression
  • What features and labels you would use
  • Where the training data would come from
  • How you would measure success
  • Potential challenges and ethical considerations

Write 200-250 words with creative, thoughtful design.

実世界の応用設計

あなたが関心のある問題（例: 教育、健康、環境、社会問題）のための教師あり学習システムを設計してください。説明してください:

  • 解決している問題
  • 分類か回帰か
  • 使用する特徴とラベル
  • 訓練データはどこから来るか
  • 成功をどのように測定するか
  • 潜在的な課題と倫理的考慮事項

創造的で思慮深い設計で200-250語で記述してください。
